# 매거진 내지 디자인 시스템 (삽만리 캐러셀 적용용)

`jeonwon_magazine_style.md` (표지)에 이어지는 내지 페이지 타입 정의.
매주 영상 1편 → 6-7장 캐러셀로 구성할 때, 아래 페이지 타입을 순서대로 조합.

---

## 공통 규칙 (모든 내지 페이지)

- **워드마크 미니 버전**: 모든 페이지 좌상단 또는 우상단 구석에 작게 "SAPMANRI" + 페이지 번호 (예: "02/07") — 표지와의 연속성, 시리즈임을 인지시킴
- **컬러**: 표지와 동일한 팔레트 (오렌지 `#E8762C`, 크림 `#FAF5EE`, 세이지 `#B2CDA8`, 다크 `#120E0C`)
- **여백**: 사진이 풀블리드(가득)인 페이지도 텍스트 영역엔 항상 충분한 여백(60px+) 확보
- **폰트**: 헤드라인 Gowun Batang Bold, 본문/캡션 Noto Sans KR Regular/Light

---

## 페이지 타입 1 — 표지 (Cover)
*이미 `jeonwon_magazine_style.md`에서 정의됨*

- SAPMANRI 워드마크 + 헤드라인 + 기사 목록 + 발행정보

---

## 페이지 타입 2 — 풀블리드 화보 + 캡션 (Full Bleed Photo)

가장 임팩트 있는 사진 1장을 페이지 전체에 채우고, 하단 또는 좌하단에 짧은 캡션 한 줄만 배치.

- **사진**: 페이지 전체 (크롭 없이 비율 유지)
- **캡션 위치**: 좌하단, 다크 그라데이션 오버레이 위에
- **캡션 텍스트**: 명조 1줄, 큰 크기 (예: "쥘리 앙드리외, 울타리를 타고 자랄 장미")
- **페이지 번호**: 우상단 작게

용도: 영상에서 가장 인상적인 한 장면 도입 (예: 직조 전등 클로즈업, 산딸기 한 그릇)

---

## 페이지 타입 3 — 인덱스/목차 (Index)

표지의 "기사 목록"을 페이지 단위로 풀어서 보여주는 목차형 페이지.

- **배경**: 다크 솔리드 또는 사진 위 어두운 오버레이
- **레이아웃**: 좌측에 작은 텍스트 "Contents" 또는 "이번 호 이야기", 우측에 번호+제목 리스트
- **리스트 항목**: "01 — 장미 울타리", "02 — 직조 전등", "03 — 산딸기", "04 — 깜짝 소식" 형태, 번호는 오렌지, 제목은 크림
- **각 항목 옆 작은 썸네일**: 정사각 크롭 이미지 (선택)

용도: 캐러셀 2번째 장 (표지 다음), "오늘 다룰 이야기" 미리보기

---

## 페이지 타입 4 — 2단 사진+텍스트 (Photo + Story Split)

좌우 또는 상하 분할: 한쪽엔 사진, 한쪽엔 짧은 글(인간극장 나레이션 발췌).

- **분할 비율**: 60:40 (사진:텍스트) 또는 반대
- **텍스트 영역**: 배경색 솔리드(다크 또는 크림), 본문 텍스트는 명조 또는 산세리프 혼용
- **텍스트 내용**: 나레이션 2-3문장 발췌, 한 문장씩 줄바꿈 (인간극장 호흡 유지)
- **작은 라벨**: 텍스트 영역 상단에 소제목 (예: "아침의 루틴")

용도: 스토리텔링이 필요한 장면 (모닝 루틴, 책 읽기, 동물 에피소드 등)

---

## 페이지 타입 5 — 그리드 콜라주 (Grid Collage)

여러 장의 사진을 격자로 배치 — 하루의 여러 순간을 한 장에 압축.

- **그리드**: 2x2 또는 1+2(큰 사진 1 + 작은 사진 2) 비대칭 그리드
- **간격**: 사진 사이 얇은 여백(8-12px), 배경은 크림 또는 다크
- **캡션**: 그리드 하단 또는 각 사진 아래 아주 작은 텍스트(선택)
- **페이지 제목**: 상단에 작은 소제목 (예: "오후의 장면들")

용도: 여러 소소한 순간들 (새 모이 주기, 카페, 풍경 등)을 한 장으로

---

## 페이지 타입 6 — 인용/하이라이트 (Quote/Highlight)

나레이션 중 가장 인상적인 한 문장을 크게 강조.

- **배경**: 솔리드 다크 또는 흐릿하게 블러된 사진
- **텍스트**: 명조 Bold, 큰 크기(48-64px), 페이지 중앙, 짧은 한 문장
- **장식**: 텍스트 위/아래에 얇은 오렌지 라인 또는 인용부호(") 모티프
- **출처/맥락**: 하단에 아주 작게 (예: "6월의 느린 하루 중")

용도: "조만간 빅뉴스를 전하게 될지도" 같은 떡밥 문장, 또는 감성적인 한 줄

---

## 페이지 타입 7 — 클로징/CTA (Closing)

캐러셀 마지막 장. 표지와 짝을 이루는 마무리 페이지.

- **배경**: 표지와 동일하거나 톤만 다른 사진
- **텍스트**: "오늘도 느리게, 잘 보냈습니다" 같은 클로징 문구 (명조, 중앙 또는 좌측)
- **CTA**: 작은 텍스트로 "유튜브에서 전체 영상 보기" + 채널 핸들 "@sapmanri"
- **워드마크**: 표지처럼 크게, 페이지 하단

용도: 캐러셀의 마지막 장, 유튜브 영상으로의 유도
