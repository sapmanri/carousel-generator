# 필름그레인(Grain) 스타일 디자인 시스템

Skyline Music Matcher `cardDesign: grain` 기준 분석. 노출 오버레이 · 빈티지 필름 · 라이트 릭.

---

## 1. 컬러 팔레트 / 효과

| 요소 | 값 |
|---|---|
| 컬러번 비네트(중심→외곽) | `transparent` → `rgba(30,18,8,0.22)` → `rgba(20,10,0,0.72)` (radial gradient) |
| 라이트 릭(좌상단) | `rgba(255,180,60,0.22)` → `rgba(255,120,30,0.08)` → `transparent` |
| 필름 그레인 노이즈 | 픽셀당 랜덤 ±30 그레이값, alpha 28, `globalCompositeOperation: overlay` |
| 곡 제목 | `rgba(255,255,255,0.95)` |
| 아티스트 | `rgba(255,255,255,0.42)` |
| 캡션 | `rgba(255,255,255,0.58)` |
| 프레임 번호 텍스트 | `rgba(255,200,80,0.55)` (앰버) |

---

## 2. 타이포그래피

- **곡 제목**: Cormorant Garamond / Noto Serif KR / Georgia, weight 300
- **아티스트**: Helvetica Neue, weight 400
- **캡션**: Cormorant Garamond, **300 italic** — 시적인 이탤릭
- **프레임 번호**: `"Courier New", monospace` — 필름 카메라의 타임코드/프레임 넘버 느낌

---

## 3. 레이아웃 / 효과 적용 순서

1. 이미지 전체에 **radial gradient 컬러번** 오버레이 (중심은 그대로, 가장자리로 갈수록 어두운 적갈색)
2. 좌상단에 **라이트 릭**(빛 번짐) 추가
3. **필름 그레인**: 캔버스 전체 크기의 노이즈 텍스처를 `overlay` 블렌드 모드로 합성
4. 파형: 이미지 하단 60~82%(세로형) / 60~88%(정방형) 영역, 흰색
5. 텍스트(하단 80% 위치):
   - 곡 제목 (세리프 라이트, 흰색 95%)
   - 아티스트 (산세리프, 흰색 42%)
   - 캡션 2줄 (이탤릭 세리프, 흰색 58%)
6. 우상단: `SAPMANRI ♦ 01A` — 모노스페이스, 앰버색 — 필름 프레임 번호처럼

---

## 4. 핵심 원칙

1. **3중 효과 레이어**: 컬러번(radial) + 라이트릭(linear) + 그레인(노이즈/overlay) — 이 순서로 합성
2. 전체 색감은 **앰버/오렌지 톤으로 따뜻하게 번짐** — 빈티지 필름 카메라의 색 누출 재현
3. 텍스트는 모두 흰색의 투명도 단계로만 — 다른 색 텍스트 없음 (프레임 번호의 앰버만 예외)
4. 모노스페이스 폰트로 된 프레임 번호가 "이것은 필름이다"라는 메타포의 핵심 디테일
5. 그레인 노이즈는 매 렌더링마다 랜덤 생성 — 매번 약간 다른 질감
