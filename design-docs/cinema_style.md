# 시네마(Cinema) 스타일 디자인 시스템

Skyline Music Matcher `cardDesign: cinema` 기준 분석. 시네마스코프 · 레터박스 · 영화적 크레딧.

## 2. 타이포그래피

- **곡 제목**: Cormorant Garamond / Noto Serif KR / Georgia, weight 300, 좌측 정렬, 크기 CW×0.038~0.042 (가장 작은 "곡 제목" — 다른 스타일보다 절제됨)
- **아티스트**: Helvetica Neue, weight 400, **uppercase**, 우측 정렬
- **캡션**: Cormorant Garamond, **300 italic**, 우측 정렬
- **채널명/화면비**: Helvetica Neue, weight 400, 매우 작음(CW×0.016)

---

## 3. 레이아웃

영화의 시네마스코프(2.39:1) 레터박스를 모사:

- **상단 레터박스 바**: 높이 = CH × 0.09(세로형) / 0.11(정방형), 솔리드 블랙
  - 좌측: `SAPMANRI` (28% 투명도)
  - 우측: `2.39:1` (화면비 표기)
  - 바 아래 얇은 구분선(6% 투명도)
- **하단 레터박스 바**: 높이 = CH × 0.14(세로형) / 0.17(정방형) — 상단보다 더 두꺼움 (텍스트 공간 확보)
  - 바 위에 얇은 구분선
  - 좌측: 곡 제목 (세리프 라이트)
  - 우측: 아티스트(uppercase) + 캡션(이탤릭)
- **파형**: 이미지 영역 최하단, 하단 바 바로 위(CH × 0.18~0.22 높이)

---

## 4. 핵심 원칙

1. **상하 레터박스(블랙 바)**로 와이드스크린 영화 비율을 흉내냄 — 가장 분명한 시각적 시그니처
2. 텍스트는 **좌(제목) / 우(아티스트·캡션) 분리 배치** — 영화 크레딧 자막 레이아웃
3. 색상은 흑백 + 투명도 단계만 — accent 컬러 없음 (Nike/Apple과 유사한 절제)
4. `2.39:1` 같은 화면비 표기, 채널명을 작은 크레딧처럼 배치하는 디테일이 "영화"라는 메타포를 강화
5. 곡 제목 크기를 의도적으로 작게(다른 스타일 대비) — 영화 엔딩 크레딧의 정적인 느낌
