# Nike 스타일 디자인 시스템

캐러셀 생성기 `style: nike` 기준 분석. 강렬·대문자·흑백 대비.

---

## 1. 컬러 팔레트

| 역할 | 색상 |
|---|---|
| 배경(메인) | `#111` (거의 블랙) |
| 배경(클로징) | `#111` solid |
| 텍스트 | `#fff` |
| 라벨/보조 텍스트 | `rgba(255,255,255,0.55)` |
| 본문 | `rgba(255,255,255,0.5)` |

→ 색상은 **흑백 단 2색**. accent 컬러를 거의 쓰지 않음 (포인트는 텍스트 자체의 크기와 대비로 만듦)

---

## 2. 타이포그래피

- 폰트: **Helvetica Neue / Arial / Noto Sans KR** (산세리프만 사용, 세리프 없음)
- 모든 헤드라인은 **대문자 변환(uppercase)** + **font-weight 700**
- 자간은 좁게(`letter-spacing: -0.01em`)

### 헤드라인 크기
| 클래스 | 크기 | line-height | 용도 |
|---|---|---|---|
| `.nike-headline` | 30px | 0.9 | 표지/피처 |
| `.nike-headline-md` | 20px | 1.0 | 분할 레이아웃 |
| `.nike-closing-hl` | 32px | 0.88 | 클로징 (가장 큼) |
| `.nike-label` | 7px | - | 라벨/마스트헤드 (자간 0.16em, uppercase) |

---

## 3. 레이아웃 / 슬라이드 타입

### 표지 (cover)
- 사진 + 하단 오버레이(`ov-bottom`)
- 마스트헤드: `SAPMANRI` (산세리프, 7px, 자간 0.2em, 흰색 30% 투명도)
- 헤드라인: `.nike-headline` 대문자 30px, 굵게

### 클로징 (closing)
- 배경: `#111` solid (사진 없이도 항상 다크)
- `.nike-closing-hl` 32px — 캐러셀 전체에서 가장 큰 텍스트
- 본문은 흰색 50% 투명도

### 인용(pull quote)
- 배경: `#111`
- `SAPMANRI` 마스트헤드(흰색 30%) + `.nike-headline` 22px 대문자 + 본문(흰색 50%)

---

## 4. 핵심 원칙

1. **모든 헤드라인 텍스트는 대문자(UPPERCASE)** — 예외 없음
2. 컬러는 **흑백만**, accent 컬러 없음 — 강렬함은 크기/굵기/대비로 표현
3. 산세리프 Bold만 사용, 세리프 절대 사용 안 함
4. 클로징 슬라이드가 가장 임팩트 있게(32px) — 캐러셀의 "마지막 펀치"
5. 자간을 좁혀(negative letter-spacing) 압축적이고 단단한 느낌
