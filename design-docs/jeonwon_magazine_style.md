# 전원생활 매거진 표지 디자인 시스템 (삽만리 적용용)

6개 표지(2022.1, 2022.2, 2022.3, 2022.5, 2022.8, 2022.9) 분석 기준

---

## 1. 레이아웃 구조

전체 레이아웃은 항상 다음 4개 영역으로 구성됨:

1. **상단 워드마크 영역** — 항상 화면 상단 중앙, 매거진명 "전원생활"
2. **메인 비주얼 영역** — 화면의 60~80%를 차지하는 인물/풍경 사진
3. **특집 텍스트 영역** — 좌상단 또는 좌중단, "특집" 라벨 + 메인 헤드라인
4. **하단 정보 바** — 좌하단 발행정보(연도/월/구독사 배지), 우하단 부가 기사 목록

### 가로 썸네일(16:9) 적용 시 변형
원본은 세로(2:3 비율) 표지이므로, 가로로 바꿀 때는:
- 워드마크: 상단 중앙 유지 (크기는 작게)
- 메인 비주얼: 화면 중앙~우측 50-70%
- 특집 텍스트: 좌측 30-40% 영역에 세로로 쌓기
- 하단 정보 바: 그대로 하단에 가로로 배치

---

## 2. 워드마크 ("전원생활" → "SAPMANRI")

- **위치**: 화면 상단 중앙, 항상 동일
- **폰트**: 두꺼운 명조체(serif), 글자 간격 넓음
- **크기**: 표지 폭의 약 35-45%
- **색상**: 핑크/레드 계열 (`#E85D6B` 유사) — 사진 배경과 무관하게 항상 같은 색
- **서브 텍스트**: 워드마크 아래 작은 텍스트 "전원의 꿈 일구는 생활정보지" (그레이, 10px급)

### 삽만리 적용
- 워드마크: "SAPMANRI" (오렌지 `#E8762C` 계열, 기존 영상 워터마크 색과 통일)
- 서브 텍스트: "오늘도 느리게 · slow days"

---

## 3. 특집 라벨 + 헤드라인

- **특집 라벨**: 작은 사각 배지 또는 단순 텍스트, "특집"이라는 2글자, 본문보다 작은 크기, 보통 빨강/오렌지 포인트 컬러
- **헤드라인**: 굵은 명조체, 2줄 구성이 많음 (예: "소소한 / 동네 이야기", "농촌 간 만은")
- **정렬**: 좌측 정렬, 줄간격 좁게 (타이트한 행간)
- **위치**: 사진의 빈 공간(주로 좌상단 또는 좌중단)에 배치, 사진 위 인물/오브젝트와 겹치지 않음

### 삽만리 적용
- 라벨: "이주의 이야기" 또는 "SAPMANRI'S PICK"
- 헤드라인: 영상 제목 2줄 (예: "내 손으로 만든 / 작은 불빛 하나")

---

## 4. 메인 비주얼

- 항상 **인물 또는 풍경이 자연광 아래** 있는 사진
- 사진 자체에 텍스트가 거의 없음 (텍스트는 사진 밖 여백에 배치)
- 톤: 채도 약간 낮춘 자연스러운 색감, 과한 필터 없음

### 삽만리 적용
- 기존 영상 스크린샷(직조 전등, 산딸기, 풍경 등) 그대로 사용
- 추가 보정 불필요 — LUT가 이미 적용된 톤이 매거진 톤과 유사함

---

## 5. 하단 정보 바

좌측부터 순서대로:
1. 작은 아이콘/배지 (예: "우수콘텐츠잡지 2022" 원형 배지)
2. **발행 연월**: 큰 숫자(연도) + 작은 숫자(월/호수), 예: "2022 | 9"
3. QR코드 + 가격 정보 (작은 텍스트)

우측:
4. **부가 기사 목록** — 2~4개의 짧은 제목, 작은 폰트, 줄바꿈으로 나열
   예: "고쳐 짓는 '사진재'", "만화가 장차헌실", "내 다리 만든 식 식모도" 등

### 삽만리 적용
- 좌측: "2026 | 6월 2주" (발행 연월 대체)
- 우측 기사 목록: 이번 영상의 소주제들
  - 장미 울타리 · 쥘리 앙드리외
  - 드디어 익은 산딸기
  - 직조 전등 만들기
  - 깜짝 소식 예고

## 7. 타이포그래피

- **워드마크**: 두꺼운 명조(세리프), 자간 넓음 — Gowun Batang Bold 또는 유사 명조 굵게
- **헤드라인**: 동일 계열 명조, 워드마크보다 작은 크기, 자간 보통
- **본문/목록**: 산세리프 가능 (Noto Sans KR Light~Regular)
- **숫자(발행연월)**: 산세리프, 큰 크기로 강조

---

## 8. 핵심 원칙 (요약)

1. 워드마크는 **항상 같은 위치, 같은 색, 같은 폰트** — 이게 "전원생활"임을 한눈에 알아보게 하는 핵심
2. 사진은 꾸미지 않고 그대로, 텍스트만 여백에 배치
3. 좌측에 "이야기"(헤드라인), 우측/하단에 "정보"(목록, 발행정보) — 정보 위계가 명확함
4. 색상은 2-3개로 제한 (워드마크색 + 텍스트색 + 보조색)
5. 행간 좁고 자간 넓은 명조 헤드라인이 "잡지"라는 인상을 만드는 핵심 요소
