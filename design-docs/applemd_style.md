# Apple (미니멀·여백) 스타일 디자인 시스템

캐러셀 생성기 `style: applemd` 기준 분석. 미니멀 · 여백 중심 · 상단 이미지/하단 블랙 패널 구조.

---

## 1. 컬러 팔레트

| 역할 | 색상 |
|---|---|
| 배경 | `#000` (순수 블랙) |
| 헤드라인 텍스트 | `#fff` |
| 라벨 | `rgba(255,255,255,0.32)` |
| 서브텍스트 | `rgba(255,255,255,0.35)` |
| 본문 | `rgba(255,255,255,0.42)` |
| 보조 본문 | `rgba(255,255,255,0.2)` |
| 룰(구분선) | `rgba(255,255,255,0.1)` |
| folio | `rgba(255,255,255,0.12)` |

→ 순수 블랙 + 흰색의 투명도 단계로만 위계 구성. accent 컬러 없음.

---

## 2. 타이포그래피

- 폰트: **Helvetica Neue / Noto Sans KR / Arial** — 산세리프만
- 헤드라인: **font-weight 200** (모든 스타일 중 가장 가벼움) — `.amd-title` 26px, line-height 1.05, letter-spacing -0.02em
- 라벨: 7px, 자간 0.22em, uppercase, 흰색 32%
- 본문: 8px, font-weight 300, **중앙 정렬**, 흰색 42%

---

## 3. 레이아웃 / 슬라이드 타입

### 일반 구조 (cover/feature/split)
- **상단 55%**: 사진 (비네트 효과: `radial-gradient(ellipse at center, transparent 40%, rgba(0,0,0,0.22) 100%)`)
- **하단 45%**: `.amd-panel` — 솔리드 블랙, 상단에 0.5px 흰 테두리(8% 투명도)
  - 라벨(`SAPMANRI` 또는 슬라이드 라벨)
  - 헤드라인 (`.amd-title`)
  - 서브텍스트(`.amd-sub`)
  - 룰(`.amd-rule`, 0.5px)
  - 본문(`.amd-body`, 중앙 정렬)

### 클로징 (closing)
- 사진 + 매우 어두운 오버레이(`rgba(0,0,0,0.85)`) 또는 솔리드 블랙
- `.amd-closing` — 중앙 정렬
  - 라벨 `SAPMANRI`
  - `.amd-title` 20px
  - 룰
  - 본문 + 보조 본문(`.amd-body2`, 채널 태그)

### 인용(pull quote) — 텍스트 전용
- `.amd-closing` 구조 재사용, 솔리드 블랙
- 룰 + 헤드라인(20px, 중앙) + 본문

---

## 4. 핵심 원칙

1. **순수 블랙(`#000`) + 흰색 투명도 단계**만 사용 — 어떤 accent 컬러도 없음
2. 헤드라인 weight는 **200** — 모든 스타일 중 가장 가볍고 미니멀
3. **상단 사진 55% / 하단 블랙 패널 45%** 구조가 모든 슬라이드의 기본 골격
4. 본문은 항상 **중앙 정렬** — 다른 스타일은 좌측 정렬이 기본인데 Apple만 중앙
5. 비네트(radial gradient)로 사진 가장자리를 살짝 어둡게 — 패널과의 경계를 부드럽게
6. 여백과 투명도 단계로만 위계를 만드는 "정제된 침묵" 같은 톤
