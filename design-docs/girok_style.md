# 기록(Girok) 스타일 디자인 시스템

Skyline Music Matcher `cardDesign: girok` 기준 분석. 라이프로그 앱 · 흰 배경 · 파스텔 포인트 · 둥근 패널.

---

## 1. 컬러 팔레트

| 역할 | 색상 |
|---|---|
| 패널 배경 | `#FFFFFF` |
| 메인 텍스트(제목) | `#1A1A1A` |
| 보조 텍스트(아티스트) | `#AAAAAA` |
| 캡션1 | `#888888` |
| 캡션2 | `#444444` |
| 캡션3 | `#BBBBBB` |
| 구분선 | `rgba(0,0,0,0.06)` / `#F0F0F5` |
| 메인 태그 pill 배경 | `#EEF0FF` |
| 메인 태그 pill 텍스트 | `#7C6FD4` (퍼플) |
| 보조 태그 pill 배경 | `#F5F5FF` |
| 보조 태그 pill 텍스트 | `#9B91E0` |
| 브랜딩 텍스트 | `#CCCCCC` |

→ 흰 배경 위에 **연한 퍼플/라벤더 계열 파스텔 포인트**가 핵심.

---

## 2. 타이포그래피

- 폰트: Noto Sans KR / Helvetica Neue / Arial — 산세리프만
- 곡 제목: **font-weight 700**, 크기 CW×0.058 (가장 굵고 큼)
- 아티스트: font-weight 400, 회색
- 캡션 3줄: 각각 300/500/300 weight로 미세한 위계 차이

---

## 3. 레이아웃

- **상단**: 이미지 풀블리드 (파형이 이미지 위 62~106% 위치에 겹쳐짐)
- **하단 패널**: 흰색, **상단 모서리만 둥글게**(border-radius ≈ CW×0.06) — 모달/바텀시트 느낌
- 패널 내부 순서(위→아래):
  1. 퍼플 pill 태그 (`line_traits[0]`)
  2. 곡 제목 (대형, bold)
  3. 아티스트 (소형, 회색)
  4. 얇은 구분선
  5. 캡션 3줄 (점층적 위계: 회색→다크→연한 회색)
  6. 우하단에 보조 파스텔 pill 태그 2개 (`line_traits[1], [2]`)
  7. 최하단 우측: `SAPMANRI` 브랜딩 (옅은 회색)

---

## 4. 핵심 원칙

1. **흰 배경 + 라벤더/퍼플 파스텔**(`#7C6FD4`, `#9B91E0`) 단일 포인트 컬러 계열
2. 모든 강조 요소는 **pill 모양**(둥근 캡슐) — 기록 앱 특유의 친근함
3. 패널은 이미지 아래 **바텀시트처럼 둥근 모서리로 얹힘**
4. 텍스트 위계가 세밀함(7단계 회색조) — 정보가 많아도 차분하게 정리되는 느낌
5. 곡 제목만 bold 700, 나머지는 모두 300~500 — 명확한 위계
