# 스테이폴리오(Stayfolio) 스타일 디자인 시스템

캐러셀 생성기 `style: stayfolio` 기준 분석. 파인스테이 · 고급스러운 여백 · 세리프 라이트.

---

## 1. 컬러 팔레트

| 역할 | 색상 | 변수명 |
|---|---|---|
| 크림 배경 | `#f7f4ef` | `--sf-cream` |
| 다크(클로징) | `#1c1917` | `--sf-dark` |
| 중간 톤 | `#7a7570` | `--sf-mid` |
| accent (브론즈) | `#8c7355` | `--sf-accent` |
| 라인 컬러 | `#d4cfc8` | `--sf-line` |

---

## 2. 타이포그래피

- **세리프**: Cormorant Garamond → Noto Serif KR (모든 헤드라인이 세리프, 매거진B보다 더 우아함)
- **산세리프**: 라벨/마스트헤드/folio에만 사용
- 헤드라인: **font-weight 300**, line-height 1.15~1.2 — 매우 가벼운 명조
- 본문: **font-style: italic**, font-weight 300, 8px, line-height 1.75 — 시적인 캡션 톤
- 라벨/마스트헤드: 6px, 자간 0.2~0.28em(매우 넓음), uppercase

### 헤드라인 크기
| 클래스 | 크기 | 용도 |
|---|---|---|
| `.sf-headline-lg` | (표지) | 표지 |
| `.sf-headline` | 24px | 클로징/일반 |
| `.sf-headline-sm` | 18px | 분할/인용 |

---

## 3. 레이아웃 / 슬라이드 타입

### 표지 (cover)
- 사진 + 그라디언트(`rgba(28,25,23,0.72)` → 투명, 더 어둡고 깊은 톤)
- 마스트헤드: `STAYFOLIO × SAPMANRI` (6px, 자간 0.26em, 흰색 45%)
- 헤드라인 `.sf-headline-lg` + 본문(흰색, 이탤릭)

### 클로징 (closing)
- 사진 + 다크 오버레이(`rgba(28,25,23,0.78)`) 또는 솔리드 `var(--sf-dark)`
- 마스트헤드(흰색 30%) + 얇은 화이트 라인(1px, 15% 투명도)
- 헤드라인 흰색 22px + 본문 + 채널 태그(accent 컬러, 자간 0.22em)

### 분할(split_bottom)
- 상단 58%: 사진
- 하단 42%: `var(--sf-cream)` 배경
  - 라벨(accent) + `.sf-headline-sm` + 본문(이탤릭, mid 컬러)

### 일반 피처
- 사진 + 그라디언트(`rgba(28,25,23,0.75)` → 투명, 52% 지점)
- 하단 텍스트: 라벨 + `.sf-headline-lg` + 흰색 본문

### 인용(pull quote) — 텍스트 전용
- 배경: `var(--sf-cream)`
- 중앙: `SAPMANRI` 마스트헤드 + 얇은 라인(`.sf-rule`) + 헤드라인(세리프) + 본문

---

## 4. 핵심 원칙

1. **모든 헤드라인이 가는 세리프(weight 300)** — Magazine B와 유사하지만 더 어둡고(`#1c1917`) 깊은 톤
2. 본문은 항상 **이탤릭** — 시적이고 사색적인 느낌 강조
3. 라벨/마스트헤드의 **자간이 매우 넓음**(0.2~0.28em) — 고급스러운 정적인 인상
4. accent(`#8c7355` 브론즈/우드톤)는 매우 절제되게, 라벨과 라인에만 사용
5. 크림(`#f7f4ef`)과 다크(`#1c1917`) 단 2개의 배경 색만 — 중간 색조 없음
