# 오늘의집(Ohouse) 스타일 디자인 시스템

캐러셀 생성기 `style: ohouse` 기준 분석. 따뜻한 라이프스타일 · 둥근 모서리 · 틸그린.

---

## 1. 컬러 팔레트

| 역할 | 색상 | 변수명 |
|---|---|---|
| 브랜드 컬러(틸그린) | `#00C896` | `--oh` |
| 브랜드 컬러 페일톤 | `#E6FAF4` | `--oh-pale` |
| 크림 배경 | `#FAFAF8` | `--oh-cream` |
| 웜 배경 | `#F5F3EF` | `--oh-warm` |

→ accent 컬러는 항상 `var(--oh)` 틸그린 단일 색상. 본문 텍스트는 다크 잉크.

---

## 2. 타이포그래피

- 폰트: Noto Sans KR / Helvetica Neue / Arial (산세리프만)
- 세리프 사용 안 함 — 친근하고 가벼운 느낌

---

## 3. 레이아웃 / 슬라이드 타입

### 표지 (cover)
- 사진 + 오버레이(`oh-ov`)
- 마스트헤드: `SAPMANRI` (`.oh-masthead`)
- 하단에 둥근 카드(`.oh-card`, border-radius 14px 14px 0 0)
  - 태그(`.oh-tag`): 틸그린 배경, 흰 텍스트, **pill 모양**(border-radius 9999px), 7px bold
  - 헤드라인(`.oh-headline-cover`)
  - 본문(`.oh-body`)

### 클로징 (closing)
- 배경: `var(--oh-pale)` (연한 민트)
- 중앙 정렬, `.oh-masthead-brand` (틸그린, bold, 자간 0.12em)
- 헤드라인 + 본문 + CTA 버튼(`.oh-cta`, 틸그린 pill 버튼)

### 일반 피처 — 분할(split_bottom / overlay_bottom)
- 상단 57%: 사진
- 하단 43%: `var(--oh-cream)` 배경 카드
  - 태그(pill), 헤드라인, 본문
  - 하단에 틸그린 라인(`.oh-line`, 2px)

### 인용(pull quote)
- `.oh-full` — `var(--oh-warm)` 배경, 중앙 정렬
- 원형 숫자 배지(`.oh-num`, 틸그린 원, 흰 숫자)
- 헤드라인 + 본문 + 하단 틸그린 라인

---

## 4. 핵심 디자인 요소

- **Pill 버튼/태그**: `border-radius: 9999px` — 모든 태그와 CTA가 완전한 둥근 캡슐 모양
- **카드형 레이아웃**: 사진 위에 둥근 모서리(14px) 카드가 얹히는 구조
- **틸그린(`#00C896`) 단일 accent** — 태그, 라인, CTA, 마스트헤드 텍스트 모두 동일 색상
- 페이지 번호(folio)는 다른 스타일보다 덜 강조됨

---

## 5. 핵심 원칙

1. 모든 강조 요소(태그, 버튼, 숫자 배지)는 **pill / 원형** — 각진 요소 없음
2. accent는 **틸그린 한 가지**로 통일, 절대 다른 색 섞지 않음
3. 배경은 항상 크림/웜 계열 — 순수 화이트나 다크 사용 안 함 (클로징의 페일 민트가 가장 어두운 배경)
4. 친근하고 따뜻한 라이프스타일 톤 — Magazine B의 절제된 톤과 대비됨
